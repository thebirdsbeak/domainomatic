Welcome!

To use the domainomatic, first load a new csv file. You can find the required format in the readme, or by checking the dummy_domains file in the assets directory.

Once a domain is loaded, you can toggle the settings and switches as appropriate. Remember to hit 'Save' for your chances to take!

Once you've entered all of the relevant settings, and have taken a screenshot, you can generate a cease and desist correspondence automatically.

For detailed guidance, see the User Guide, which is available offline via the help menu.
